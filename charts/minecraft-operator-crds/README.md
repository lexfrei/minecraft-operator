[![Release](https://img.shields.io/github/v/release/lexfrei/minecraft-operator)](https://github.com/lexfrei/minecraft-operator/releases)

Custom Resource Definitions for Minecraft Operator

## Overview

This chart installs Custom Resource Definitions (CRDs) for Minecraft Operator:

- **PaperMCServer** — Defines a Minecraft server instance
- **Plugin** — Defines a plugin for matched servers

CRDs are installed separately for independent lifecycle management. Helm does not automatically upgrade CRDs, so this chart allows you to manage them explicitly.

## Installation

```bash
helm install minecraft-operator-crds \
  oci://ghcr.io/lexfrei/minecraft-operator-crds \
  --version 0.4.0 \
  --namespace minecraft-operator-system \
  --create-namespace
```

## Upgrade

```bash
helm upgrade minecraft-operator-crds \
  oci://ghcr.io/lexfrei/minecraft-operator-crds \
  --version NEW_VERSION \
  --namespace minecraft-operator-system
```

## Uninstall

> **Warning:** Uninstalling CRDs will delete all PaperMCServer and Plugin custom resources!

```bash
helm uninstall minecraft-operator-crds \
  --namespace minecraft-operator-system
```
